# Piano samples for RiffScore / Tone.js

RiffScore’s built-in piano uses `Tone.Sampler` with files under `/audio/piano/` (see `riffscore` package). These **Salamander Grand Piano** samples are hosted by the Tone.js project for demos.

**Source:** [tonejs.github.io/audio/salamander](https://github.com/Tonejs/audio/tree/gh-pages/salamander) — Yamaha C5; CC BY 3.0 where applicable; see upstream for full license text.

**Populate locally** (requires network):

```bash
./scripts/fetch-riffscore-piano-samples.sh
```

Committed checkouts may include the `.mp3` files so `npm run dev` works offline.
